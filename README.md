Activity Title; Introduction to Python, Git, and Github

## Reflection

Git and GitHub are useful for programming students because they help track changes in code and make it easy to manage different versions of projects. They also support collaboration, allowing multiple students to work on the same project and share their code efficiently. Learning Git and GitHub prepares students for professional software development practices used in the industry.
